Проект диплома по созданию сайта интернет-магазина v 1.00.

Цель проекта: Создать рабочую версию сайта интернет магазина и задеплоить на хостинге.

Задачи проекта:
1) Прокачать навыки frontend development и code review.
2) Отработать навыки работы в команде.
3) Не утопить в говнокоде единственного back-end developer проекта.

ПРАВИЛА написания кода в ракмках проекта:
1) Разработка ведется только в ветке dev репозитория проекта(комитить в ветке main ЗАПРЕЩЕНО).
2) ПЕРЕД началом разработки функционала нужно создать задачу на доске проекта на сайте TRELLO и убедится что эта задача еще никем не разрабатывается.
3) Разработка JS кода идет только в папке js путем создания отдельных js файлов подключаемых к соостветсятвующим html страницам. ПИСАТЬ JS КОД НЕПОСРЕДСТВЕННО В HTML ФАЙЛЕ ЗАПРЕЩЕНО!!!
4) Функционал относящийся ко всем страницам проекта предлагается экспортировать в общий файл script.js.
5) Разработка ведется по принципам обьект ориентированного программирования(процедурный подход НЕИСПОЛЬЗОВАТЬ!).
6) Приветствуется создание модулей в папке js/modules/ проекта.
7) Обязательно написание документации для созданного Вами функционала с инструкцими по логике и использованию(место расположения документации в проработке).
8) При создании js файла содержащего класс файл называется именем соответствующим классу и название начинается с заглавной буквы.
9) При создании выбранного разработчиком функционала создается дочерняя ветка от dev с названием feature-название-фичи (название пишется только через дефис).
10) При создании функционала карточка с задачей разработчика должна быть перемещена в состояние in progress.
11) Когда функционал закончен карточка задачи перемещается в состояние testing.
12) Слияние с веткой dev допускается только после проведения code review и одобрения функционала проверяющими код и перемешения карточки задачи в статус done.
13) При выявлении багов функционал возвращается на доработку и повторное тестирование о чем уведомляется разработчик функционала.
14) Взаимодействие разработчиков осуществляется путем очного общения или сообщений в Telegram группе Диплом front.
